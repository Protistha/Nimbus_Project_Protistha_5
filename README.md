# Lib_Book_Management_System
C Project for managing library books.
Overview: This project is a console-based application created to aid librarians to manage issued book records and organize books efficiently. 
Features:
- Add book-
-  Delete book
- Display list of books
- Issue book
- Display list of issued books
- Cross platform screen clearing (cls for Windows, clear for Linux/Unix).
Prerequisites
- GCC or any C compiler
- A terminal/command prompt
Usage steps-
- After installing a C compiler, compile the program via the following commands:
For Windows(PowerShell / CMD)
gcc library.c -o library
For Linux/macOS
gcc library.c -o library
-Run the program
For Windows
library.exe
For Linux/ macOS
./library
=> When the program starts, you will see a menu:
<<----- Library Book Management System ----->>

1. Issue Book
2. Add New Book
3. Delete Book
4. Display Books List
5. Issued Books List
0. Exit
